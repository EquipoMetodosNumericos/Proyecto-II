Métodos Numéricos II

Unidad I: Introducción a la Solución de Sistemas de Ecuaciones No Lineales

Esta unidad está dedicada al estudio de los sistemas de ecuaciones no lineales, los cuales constituyen una herramienta fundamental en la modelación de fenómenos complejos en diversas áreas del conocimiento, como la ingeniería, la física, la química, la economía, entre otras.

A diferencia de los sistemas lineales, los sistemas no lineales no se pueden resolver utilizando técnicas algebraicas simples, ya que sus soluciones pueden no ser únicas, no existir o depender fuertemente de las condiciones iniciales. Por esta razón, se recurre a métodos numéricos que permiten aproximar las soluciones con cierto grado de precisión.

Entre los métodos más destacados se encuentra el método de Newton-Raphson, que utiliza derivadas para encontrar sucesivas aproximaciones de las raíces del sistema, y el método de punto fijo, que transforma el sistema original en una forma iterativa. Se analizan las condiciones necesarias para garantizar la convergencia de estos métodos y se estudian sus ventajas y limitaciones.

Además, se introduce el uso de herramientas computacionales como software de cálculo simbólico y lenguajes de programación numérica para implementar algoritmos que permiten resolver estos sistemas de manera eficiente y práctica.

Puntos principales:

  • Identificar y formular sistemas de ecuaciones no lineales a partir de modelos reales.
  • Aplicar métodos iterativos para encontrar soluciones aproximadas.
  • Analizar la convergencia y estabilidad de los métodos utilizados.
  • Implementar soluciones computacionales para resolver problemas no lineales.

MENÚ UNIDAD I






Introducción Método de Punto Fijo

El método de punto fijo es una técnica iterativa utilizada para resolver sistemas de ecuaciones no lineales. Consiste en transformar el sistema original en una forma equivalente:

\( x = g(x) \)

Donde g(x) es una función vectorial que define la iteración. A partir de una estimación inicial x0, se genera una sucesión de aproximaciones:

\( x_{n+1} = g(x_n) \)

Pasos generales del método

  1. Reescribir el sistema como x = g(x).
  2. Elegir una estimación inicial x0.
  3. Iterar usando la relación xn+1 = g(xn).
  4. Detener cuando la diferencia entre iteraciones sea menor que una tolerancia dada.

Condiciones de convergencia

Para que el método converja, la función g(x) debe ser contractiva en el intervalo de interés. Una condición suficiente es que el norma del Jacobiano de g sea menor que 1:

\( J_{g(x)} < 1 \)

Ventajas y desventajas

  • Ventajas: Fácil de implementar, útil para obtener una primera aproximación.
  • Desventajas: Requiere transformación adecuada del sistema, puede no converger si no se cumple la condición de contracción.







MENÚ PUNTO FIJO






Método de Newton-Raphson

El método de Newton-Raphson es una técnica iterativa utilizada para resolver sistemas de ecuaciones no lineales de la forma:

F(x) = 0

Donde F(x) es una función vectorial. A partir de una estimación inicial x0, se calcula una sucesión de aproximaciones mediante:

xn+1 = xn - J-1(xn) · F(xn)

Donde J(x) es el Jacobiano de F(x), es decir, la matriz de derivadas parciales.

Pasos generales del método

  1. Definir el sistema F(x) = 0.
  2. Calcular el Jacobiano J(x).
  3. Elegir una estimación inicial x0.
  4. Iterar usando la fórmula de Newton:

    5. xn+1 = xn - J-1(xn) · F(xn)

  5. Detener cuando ||xn+1 - xn|| sea menor que una tolerancia.

Ventajas y desventajas

  • Ventajas: Alta velocidad de convergencia (cuadrática), muy preciso si está cerca de la raíz.
  • Desventajas: Requiere el cálculo y la inversión del Jacobiano, puede no converger si la estimación inicial no es adecuada.

Recomendaciones

Es preferible utilizar este método cuando se tiene una buena aproximación inicial y se puede calcular el Jacobiano de manera eficiente.
