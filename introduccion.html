UNIDAD I

Resolución Numérica de Sistemas de Ecuaciones No Lineales

Los sistemas de ecuaciones no lineales son comunes en modelado matemático y requieren métodos numéricos para su solución cuando no es posible resolverlos analíticamente.

Método de Punto Fijo

También conocido como método de sustitución sucesiva, transforma el sistema original en uno equivalente:

\[ \mathbf{x} = \mathbf{G}(\mathbf{x}) \]

donde \(\mathbf{x} = (x_1, x_2, \dots, x_n)^T\) y \(\mathbf{G}\) es la función de iteración.

Algoritmo Iterativo

\[ \mathbf{x}^{(k+1)} = \mathbf{G}(\mathbf{x}^{(k)}) \]

Condición de Convergencia

\[ \|\mathbf{G}'(\mathbf{x})\| < 1 \quad \text{en una vecindad de la solución} \]

Ventajas

  • Simple de implementar
  • No requiere cálculo de derivadas
  • Puede converger rápidamente si \(\mathbf{G}\) está bien elegida

Desventajas

  • La convergencia no está garantizada
  • La elección de \(\mathbf{G}\) es crucial
  • Puede converger lentamente

Método de Newton-Raphson

Método basado en la linealización del sistema mediante series de Taylor:

\[ \mathbf{F}(\mathbf{x}) = \mathbf{0} \]

donde \(\mathbf{F} = (f_1, f_2, \dots, f_n)^T\) es el sistema original.

Algoritmo Iterativo

\[ \mathbf{J}(\mathbf{x}^{(k)}) \Delta\mathbf{x}^{(k)} = -\mathbf{F}(\mathbf{x}^{(k)}) \] \[ \mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} + \Delta\mathbf{x}^{(k)} \]

donde \(\mathbf{J}\) es la matriz jacobiana:

\[ J_{ij} = \frac{\partial f_i}{\partial x_j} \]

Condición de Convergencia

\[ \mathbf{J}(\mathbf{x}^*) \text{ debe ser no singular en la solución } \mathbf{x}^* \]

Ventajas

  • Convergencia cuadrática cerca de la solución
  • Eficiente para sistemas con buen comportamiento
  • Precisión elevada

Desventajas

  • Requiere cálculo del jacobiano
  • Necesita resolver un sistema lineal en cada iteración
  • Sensible a la aproximación inicial

Comparación de Métodos

Aspecto Punto Fijo Newton-Raphson
Convergencia Lineal (lenta) Cuadrática (rápida)
Cálculo de derivadas No requiere Requiere jacobiano
Complejidad Baja Alta
Estabilidad Depende de \(\mathbf{G}\) Depende de aproximación inicial
Aplicaciones Sistemas simples Sistemas complejos

Recomendaciones

  • Usar Punto Fijo cuando el sistema puede transformarse fácilmente y se espera convergencia
  • Usar Newton-Raphson cuando se necesita alta precisión y se dispone de recursos computacionales
  • Considerar métodos híbridos o modificaciones (Newton simplificado, Broyden) para casos intermedios
UNIDAD II

Interpolación y Aproximación Polinomial

Técnicas para construir funciones polinomiales que representen conjuntos de datos discretos, ya sea pasando exactamente por los puntos (interpolación) o aproximando la tendencia general (aproximación).

Interpolación de Lagrange

Fórmula

\[ P(x) = \sum_{i=0}^{n} y_i \cdot L_i(x) \] \[ L_i(x) = \prod_{\substack{j=0 \\ j \ne i}}^{n} \frac{x - x_j}{x_i - x_j} \]

Características

  • Interpolación exacta: Pasa por todos los puntos dados
  • Polinomio único de grado ≤ n para n+1 puntos
  • No requiere resolver sistemas de ecuaciones

Ventajas

  • Fácil implementación conceptual
  • Estable para pocos puntos
  • Útil cuando no se pueden añadir nuevos puntos

Desventajas

  • Costoso computacionalmente para muchos puntos
  • Difícil actualización al añadir nuevos puntos
  • Fenómeno de Runge con puntos equidistantes

Diferencias Divididas de Newton

Fórmula

\[ P(x) = f[x_0] + f[x_0,x_1](x-x_0) + \cdots + f[x_0,\ldots,x_n]\prod_{i=0}^{n-1}(x-x_i) \] \[ f[x_i,\ldots,x_{i+k}] = \frac{f[x_{i+1},\ldots,x_{i+k}] - f[x_i,\ldots,x_{i+k-1}]}{x_{i+k}-x_i} \]

Características

  • Interpolación exacta como Lagrange
  • Forma anidada eficiente para evaluación
  • Permite actualización incremental

Ventajas

  • Más eficiente que Lagrange
  • Fácil añadir nuevos puntos
  • Forma adecuada para implementación computacional

Desventajas

  • Sensible al orden de los puntos
  • Requiere construcción de tabla de diferencias
  • Mismos problemas de oscilación que Lagrange

Mínimos Cuadrados

Formulación

\[ \min \sum_{i=1}^m (y_i - P(x_i))^2 \] \[ P(x) = a_0 + a_1x + \cdots + a_nx^n \]

Sistema Normal

\[ \mathbf{A}^T\mathbf{A}\mathbf{a} = \mathbf{A}^T\mathbf{y} \]

donde \(A_{ij} = x_i^{j-1}\)

Características

  • Aproximación: No pasa necesariamente por los puntos
  • Minimiza el error cuadrático medio
  • Útil para datos con ruido o errores

Ventajas

  • Robusto frente a errores en los datos
  • Permite controlar el grado del polinomio
  • Ampliamente usado en estadística y ciencia de datos

Desventajas

  • Solución aproximada, no exacta
  • Sistema normal puede ser mal condicionado
  • Elección del grado es crítica

Comparación y Aplicaciones

Aspecto Lagrange Diferencias Divididas Mínimos Cuadrados
Tipo Interpolación exacta Interpolación exacta Aproximación
Grado Polinomio n (para n+1 puntos) n (para n+1 puntos) Se elige (generalmente << n)
Eficiencia O(n²) operaciones O(n²) pero más eficiente O(mn²) (m puntos, grado n)

Recomendaciones de Uso

Usar Lagrange cuando:
  • Se requiere exactitud en puntos específicos
  • El número de puntos es pequeño
  • No se añadirán nuevos puntos
  • Implementación simple es prioridad
Usar Diferencias Divididas cuando:
  • Se necesita eficiencia computacional
  • Posible adición de nuevos puntos
  • Evaluación frecuente del polinomio
  • Interpolación exacta es requerida
Usar Mínimos Cuadrados cuando:
  • Los datos tienen ruido o errores
  • Se busca tendencia general, no exactitud
  • Hay muchos puntos de datos
  • Prevención de sobreajuste es importante

Consideraciones Importantes

Fenómeno de Runge

En interpolación polinomial con puntos equidistantes, al aumentar el grado pueden aparecer oscilaciones grandes en los extremos del intervalo. Soluciones:

  • Usar puntos de Chebyshev para la interpolación
  • Emplear interpolación por splines
  • Considerar aproximación por mínimos cuadrados

Elección del grado en Mínimos Cuadrados

Un grado demasiado alto puede llevar a sobreajuste (overfitting), mientras que uno demasiado bajo puede no capturar la tendencia. Técnicas para seleccionar el grado óptimo:

  • Validación cruzada
  • Análisis de residuos
  • Criterios de información (AIC, BIC)

Métodos de Integración Numérica

Aproximación del área bajo la curva para integrales definidas

Regla del Trapecio

Fórmula Simple
\[ \int_a^b f(x)dx \approx \frac{h}{2}[f(a) + f(b)] \]

donde \( h = b - a \)

Fórmula Compuesta
\[ \int_a^b f(x)dx \approx \frac{h}{2}\left[f(a) + 2\sum_{i=1}^{n-1}f(x_i) + f(b)\right] \]

con \( h = \frac{b-a}{n} \), \( x_i = a + ih \)

Error de Truncamiento

\( E_T = -\frac{(b-a)^3}{12n^2}f''(\xi) \), \( \xi \in [a,b] \)

Características:
  • Aproxima con rectas (polinomios de grado 1)
  • Error proporcional a \( h^2 \)
  • Exacta para polinomios de grado ≤ 1

Regla de Simpson 1/3

Fórmula Simple
\[ \int_a^b f(x)dx \approx \frac{h}{3}[f(a) + 4f(c) + f(b)] \]

donde \( h = \frac{b-a}{2} \), \( c = \frac{a+b}{2} \)

Fórmula Compuesta
\[ \int_a^b f(x)dx \approx \frac{h}{3}\left[f(a) + 4\sum_{\substack{i=1 \\ i \text{ impar}}}^{n-1}f(x_i) + 2\sum_{\substack{i=2 \\ i \text{ par}}}^{n-2}f(x_i) + f(b)\right] \]

con \( n \) par, \( h = \frac{b-a}{n} \)

Error de Truncamiento

\( E_S = -\frac{(b-a)^5}{180n^4}f^{(4)}(\xi) \), \( \xi \in [a,b] \)

Características:
  • Aproxima con parábolas (polinomios de grado 2)
  • Error proporcional a \( h^4 \)
  • Exacta para polinomios de grado ≤ 3
  • Requiere número par de intervalos

Regla de Simpson 3/8

Fórmula Simple
\[ \int_a^b f(x)dx \approx \frac{3h}{8}[f(a) + 3f(a+h) + 3f(a+2h) + f(b)] \]

donde \( h = \frac{b-a}{3} \)

Fórmula Compuesta
\[ \int_a^b f(x)dx \approx \frac{3h}{8}\left[f(a) + 3\sum_{\substack{i=1 \\ i \not\equiv 0 \mod 3}}^{n-1}f(x_i) + 2\sum_{\substack{i=3 \\ i \equiv 0 \mod 3}}^{n-3}f(x_i) + f(b)\right] \]

con \( n \) múltiplo de 3, \( h = \frac{b-a}{n} \)

Error de Truncamiento

\( E_{S38} = -\frac{(b-a)^5}{6480n^4}f^{(4)}(\xi) \), \( \xi \in [a,b] \)

Características:
  • Aproxima con polinomios cúbicos (grado 3)
  • Error proporcional a \( h^4 \)
  • Exacta para polinomios de grado ≤ 3
  • Requiere número de intervalos múltiplo de 3

Comparación de Métodos

Grado Polinomio 1 (lineal) 2 (cuadrático) 3 (cúbico)

Ejemplo Práctico

Calcular \( \int_0^1 e^{x^2} dx \) (sin primitiva elemental) con n=6:

Trapecio Compuesto

h = 1/6 ≈ 0.1667

Resultado ≈ 1.4672

Error estimado: O(10⁻³)

Simpson 1/3

h = 1/6 ≈ 0.1667

Resultado ≈ 1.4627

Error estimado: O(10⁻⁵)

Simpson 3/8

h = 1/6 ≈ 0.1667

Resultado ≈ 1.4627

Error estimado: O(10⁻⁵)

Valor real aproximado: 1.462651745907181...

Recomendaciones Prácticas

  1. Para resultados rápidos: Usar Trapecio compuesto con n moderado
  2. Para máxima precisión: Simpson 1/3 con n adecuado
  3. Cuando n es múltiplo de 3: Considerar Simpson 3/8
  4. Para funciones periódicas: Trapecio puede ser sorprendentemente bueno
  5. Adaptabilidad: Combinar métodos según el intervalo

Consideraciones Importantes

  • El error depende de las derivadas de f(x)
  • Simpson requiere continuidad de f⁽⁴⁾(x) para el término de error
  • Para funciones con singularidades, considerar métodos especiales
  • En implementaciones computacionales, usar la forma anidada para evaluar polinomios
  • Para alta precisión, considerar cuadratura gaussiana
