Unidad II: Interpolación y Aproximación Polinomial

En esta unidad se abordan los fundamentos teóricos y prácticos de la interpolación y la aproximación polinomial, herramientas esenciales en el análisis numérico para estimar valores desconocidos a partir de datos discretos.

La interpolación consiste en encontrar una función polinómica que pase exactamente por un conjunto de puntos dados. Este método es útil cuando se tienen datos experimentales o tabulados y se requiere construir una función que modele el comportamiento de dichos datos. Se exploran métodos clásicos como la interpolación de Lagrange y la interpolación de Newton, analizando sus ventajas, desventajas y eficiencia computacional.

Por otro lado, la aproximación polinomial busca construir una función que no necesariamente pase por todos los puntos, pero que minimice el error global entre los datos y la función. En este contexto, se introduce el concepto de mínimos cuadrados y su aplicación en la regresión polinomial, una técnica ampliamente utilizada en estadística y procesamiento de datos.

También se discute el problema del overfitting al utilizar polinomios de alto grado, y se analiza el teorema de interpolación de Weierstrass que garantiza la existencia de una aproximación polinómica para funciones continuas.

Finalmente, se realiza la implementación de los algoritmos en herramientas computacionales para visualizar gráficamente los resultados y comparar el comportamiento de diferentes métodos.

Puntos principales:

  • Aplicar métodos de interpolación para construir funciones polinómicas que ajusten datos discretos.
  • Usar técnicas de aproximación por mínimos cuadrados para modelar relaciones funcionales aproximadas.
  • Evaluar el error de interpolación y la calidad de una aproximación polinomial.
  • Implementar soluciones numéricas usando software como Python o MATLAB para visualizar resultados.

Unidad II: Interpolación y Ajuste

Interpolación de Lagrange

La interpolación de Lagrange permite construir un polinomio que pasa exactamente por un conjunto de puntos dados. Es muy útil cuando se conoce un conjunto de datos discretos y se desea estimar el valor de la función en un punto intermedio.

Fórmula del Polinomio de Lagrange

\[ P(x) = \sum_{i=0}^{n} y_i \cdot L_i(x) \]
\[ L_i(x) = \prod_{\substack{j=0 \\ j \ne i}}^{n} \frac{x - x_j}{x_i - x_j} \]

Donde \(L_i(x)\) son los polinomios base, construidos para que \(L_i(x_j) = 1\) si \(i = j\) y 0 en otro caso.

Ventajas

  • Fácil de implementar.
  • No requiere resolver sistemas de ecuaciones.

Desventajas

  • Difícil de actualizar al añadir nuevos puntos.
  • Puede presentar oscilaciones si hay muchos puntos.

¿Para qué se usa?

  • Interpolar funciones a partir de puntos conocidos.
  • Estimar valores intermedios en tablas.
  • Aplicaciones en gráficos, ingeniería y análisis numérico.

Método de Diferencias Divididas de Newton

El método de diferencias divididas de Newton permite encontrar el polinomio interpolante que pasa por un conjunto de puntos \((x_0, y_0), (x_1, y_1), \dots, (x_n, y_n)\) de forma eficiente. A diferencia de Lagrange, este método genera una fórmula que se puede actualizar fácilmente si se agregan más puntos.

Fórmula del Polinomio de Newton

\[ P(x) = f[x_0] + f[x_0,x_1](x - x_0) + f[x_0,x_1,x_2](x - x_0)(x - x_1) + \dots + f[x_0,\dots,x_n](x - x_0)\dots(x - x_{n-1}) \]

Donde \(f[x_i]\), \(f[x_i,x_{i+1}]\), etc., son las diferencias divididas, calculadas recursivamente con:

\[ f[x_i, x_{i+1}] = \frac{f[x_{i+1}] - f[x_i]}{x_{i+1} - x_i} \] \[ f[x_i, x_{i+1}, x_{i+2}] = \frac{f[x_{i+1}, x_{i+2}] - f[x_i, x_{i+1}]}{x_{i+2} - x_i} \] Y así sucesivamente.

Ventajas

  • Fácil de actualizar al agregar nuevos puntos.
  • Menor costo computacional que Lagrange para muchos puntos.

Desventajas

  • Puede ser laborioso calcular las diferencias si no se usa una tabla.
  • El orden de los puntos afecta la forma del polinomio.

¿Dónde se usa?

  • Interpolación numérica de funciones.
  • Construcción de tablas y aproximaciones.
  • Evaluación rápida con forma escalonada (Horner).

Ajuste por Mínimos Cuadrados

El método de mínimos cuadrados se usa para encontrar una función (normalmente una recta o una parábola) que se aproxime lo mejor posible a un conjunto de datos que no necesariamente cumplen con un comportamiento exacto.

Objetivo

Minimizar la suma de los cuadrados de las diferencias entre los valores reales \(y_i\) y los valores aproximados \(\hat{y}_i\):

\[ S = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

Recta de Ajuste (modelo lineal)

Para ajustar una recta de la forma \(y = a_0 + a_1 x\), los coeficientes se obtienen resolviendo el sistema:

\[ a_0 n + a_1 \sum x_i = \sum y_i \] \[ a_0 \sum x_i + a_1 \sum x_i^2 = \sum x_i y_i \]

Una vez calculados \(a_0\) y \(a_1\), se tiene la recta que mejor se ajusta a los datos según el criterio de mínimos cuadrados.

Ventajas

  • Ideal cuando los datos tienen ruido o no siguen un patrón exacto.
  • Fácil de generalizar a curvas cuadráticas, cúbicas, etc.

Desventajas

  • El resultado es una aproximación, no pasa por todos los puntos.
  • Puede no ser adecuado si el modelo elegido no representa bien los datos.

Aplicaciones comunes

  • Ajuste de curvas en estadísticas y ciencia de datos.
  • Predicción de tendencias lineales o no lineales.
  • Procesamiento de señales y análisis experimental.
